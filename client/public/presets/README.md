# Maestro Preset Audio Directory

You can place your `.mp3` audio files in this directory to load your own full tracks offline:

1. **`kesariya.mp3`**
2. **`pashmina.mp3`**
3. **`yeh_fitoor_mera.mp3`**

If no local file is found in this directory, Maestro will automatically fetch a dynamic 30-second song preview from the iTunes Search API, making sure your Quick Demo Tracks always work out-of-the-box!
